# Utilities to Get JSON Codes from .NET/C# Codebase

Developed with:

- Node.js
- Bash
- AWK
- Docker
- .NET/C#

## How To Use It

Run the solution with **Docker** on Linux/Unix systems with:

```bash
make run
```

OR, locally, on Linux (only tested on Fedora 36):

```bash
make local
```

> The generated files will be located at `assets` directory
